~/devenv-setup

Automation scripts for new local development environments

Linux (Ubuntu/Arch/Fedora)

Bootstrap your Linux environment (Debian, Ubuntu, Arch, CachyOS, or Fedora) with a single command.

bash
$curl -sSL https://raw.githubusercontent.com/BryanSant/devenv/main/devenv.main.sh | bash

macOS

Setup Homebrew, essential development tools, and your shell environment.

zsh
%curl -sSL https://raw.githubusercontent.com/BryanSant/devenv/main/devenv.main.sh | bash

Extra Tools

Install Docker, GitHub CLI, Gemini, Firebase, Terraform, k3d, Skaffold, and Google Cloud CLI.

bash
$curl -sSL https://raw.githubusercontent.com/BryanSant/devenv/main/devenv.extra.sh | bash
